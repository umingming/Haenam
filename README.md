# 📖 Introduction
### 1. 프로젝트 개요
* 해냄은 일정을 작성하고 모두 해내자는 의미입니다.
* 평소 작성하는 스케줄러에서 아이디어를 얻어 Node.js와 Mongo db를 활용해 개발했습니다. 
<br />

### 2. 개발 환경
* vue3
* Node.js v18.16.0
* MongoDB
<br />

### 3. 프로젝트 내용
#### 3-1. 로그인 및 회원가입
<img src="https://github.com/umingming/Haenam/assets/87955005/98c95f2c-01a2-4671-9a57-522e439bc70d" alt="image" width="50%"><br />
* 버튼 선택에 따라 로그인, 회원가입 전환
* 로그인 상태 유지 가능
<br />

#### 3-2. 메인
<img src="https://github.com/umingming/Haenam/assets/87955005/58705b6a-d9ce-457b-8ab7-68086d431d6d" alt="image" width="50%"><br />

* 더블 클릭으로 내용 수정 및 삭제 가능
